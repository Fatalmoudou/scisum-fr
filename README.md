# SciSum-FR

SciSum-FR est un outil open-source de résumé automatique d’articles
scientifiques en français.

## Objectifs
- Constituer un corpus scientifique français
- Nettoyer et structurer les textes
- Analyser et regrouper les documents
- Générer des résumés automatiques
- Fournir un logiciel installable et documenté
